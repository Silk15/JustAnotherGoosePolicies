# Just Another Goose (Privacy Policy)
**Last Updated: June 10th 2026**

This bot collects data about your server's users via (but not limited to) the following methods; Forms, Bans, Punishments, Modums, Command uses. You retain the right to access and request deletion of this saved data at any time!

What I collect:
- User IDs
- Role IDs
- Channel IDs
- Form responses
- Punishment durations/reasons
- Personal server IPs and ports (If using the server module)

What I don't collect:
- Email addresses or any other personal info (Such as tokens)
- Chat history, statistics or info
- Engagement data or information on personal biases

All data is stored locally on a personal machine and is not uploaded to any form of cloud, nor is this data sold/modifed by anyone other than me and my bot. As stated above you are completely entitled to request deletion of this data should you want to remove it.
